Esta es mi primera practica con Git y GitHub

En esta práctica realizaremos todo lo visto en las primeras 20 clases del curso "Curso Profesional de Git y GitHub" de Platzi que dicto Freddy Vega.

Como pudimos ver Git es un sistema de control de versiones de forma distribuida. Que nos permite trabajar un proyecto de forma ordenada sin perder información. Puesto que nos crea un listado de todo lo que se realizó de manera históricaa. Por su parte GitHub es una plataforma online que nos permite subir nuestro código a un repositorio online para poder trabajar en grupo.
